Overview
================
This module provides integration with bxSlider

Features
===========
Carrousel integration with Views & Views slideshow
BxSlider integration

Links to useful resources & Requirements
=============
Drupal 7
Views Slideshow
Libraries
BxSlider

Keywords
===========

bxslider
carrousel, carousel
slideshow
views slideshow

How to
================
Install the required modules
Copy bxslider package you downloaded
from the official site : /sites/all/libraries/bxslider (if bxslider doesn't
 exist create it, and that you have :
  /sites/all/libraries/bxslider/jquery.bxslider.min.js)

Call backs
=================
Call backs cannot be used on the admin preview screen
