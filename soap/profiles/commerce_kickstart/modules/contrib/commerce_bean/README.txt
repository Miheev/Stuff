Integration of Drupal Commerce and Bean
=======================================
This module exposes Beans as checkout panes for drupal commerce checkout
process.

Installation and settings
=========================
Install module as usual (Drupal Commerce and Bean are required) and configure
which bean types will expose beans in the checkout inteface in
Administer > Store > Configuration > Commerce bean

Drag and drop the beans you want in each checkout step in the checkout form
Administer > Store > Configuration > Commerce bean
You can select the view mode for each bean.
